# Dodgeball-Survival-Game

Neon Dodgeball Walkthrough
Overview
Successfully implemented a dynamic, web-based dodgeball game using HTML, CSS, and the Canvas API in JavaScript. The game features a modern UI with glassmorphism style, neon visual effects, dynamic difficulty scaling, and mobile touch support.

Files Created
index.html
 - Contains the main layout, UI screens (Start and Game Over), scoreboard, and the interactive canvas element.
style.css
 - Includes the styling, animations, interactive button behaviors, and the neon/glassmorphism aesthetic.
script.js
 - Handles the core game loop, rendering elements on the canvas, input event listeners (keyboard and touch), and the collision detection engine.
Key Features
Visuals: Implemented a dark theme with glowing neon objects, dynamic shadow blurring using Canvas Context, and a transparent trailing effect for motion blur.
Controls: Event listeners for ArrowLeft and ArrowRight arrow keys for desktop, and touch/drag support for mobile environments.
Dynamic Difficulty: Balls spawn at random sizes and increasing speeds based on the player's current score. The spawn frequency also gradually increases as the game progresses.
Responsive Canvas: The HTML Canvas dynamically resizes to fit the container constraints cleanly.
How to Play
Open the 
index.html
 file in your web browser.
Click the "Start Game" button to start.
Use your Left and Right arrow keys to dodge the colorful balls falling from the top.
If you are hit, the final score will be displayed, and you can easily restart by clicking "Restart".
